# Financial Sentiment Analysis

## Project Overview
Analysis of correlations between financial news sentiment and stock market movements for Nova Financial Insights.

## Project Structure
financial-sentiment-analysis/
├── .vscode/ # VS Code settings
├── .github/ # GitHub workflows

├── notebooks/ # Jupyter notebooks for analysis
├── tests/ # Test files

└── data/ # Data directory

- Task 1: EDA & Statistical Analysis
- Task 2: Quantitative Analysis with PyNance and TA-Lib
